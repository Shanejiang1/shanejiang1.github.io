如何使用hugo搭建个人博客

1 安装hugo [官方教程] (https://gohugo.io/getting-started/installing)

1.1 [huog releases页面] (https://github.com/gohugoio/hugo/releases) 下载hugo_x.xx.x_Windows-64bit.zip

1.2 解压，把hugo解压到D:\Software\hugo

1.3 把D:\Software\hugo 添加到PATH

1.4 重启终端，运行hugo version查看版本

2 进入一个你想存放个人博客的文件夹，创建一个新的hugo网站

2.1 注意：xxxxxx代表你的github用户名，这里用户名全部使用小写。

2.2 这里以我的设置为例 hugo new site xxxxxx.github.io-creator

3 添加主题

3.1 进入这个目录 cd xxxxxx.github.io-creator

3.2 git init

3.3 这里以我的主题为例 git submodule add https://github.com/softwareyoga/ronu-hugo-theme.git themes/ronu-hugo-theme

3.4 在config.toml文件中指定作为默认主题,只需添加行 theme = "ronu-hugo-theme"

4 config.toml 基本配置

4.1 首先在配置文件中包含以下配置

baseURL = “ https://www.example.com/ ”
 
 #CN改为大写
 languageCode = “ zh-CN ”
 
 title = “ MySiteTitle ”
 
 theme = “ ronu-hugo-theme ”

5 开始写文章

5.1 hugo new posts/my-first-post.md

5.2 写博客请在以下内容下面开始

---
title: "My First Post"

date: 2019-03-26T08:47:11+01:00
# 这里把true改成false
draft: true
---

6 启动hugo服务器

6.1 当博客写好后，启动终端，运行hugo server -D

6.2 在运行的下方找到http://localhost:1313/点击进入就能看到博客了

7 建立静态页面

7.1 终端运行hugo后会创建一个新的目录叫public

如何把public上传到github

1 在xxxxxx.github.io-creator目录里新建一个文件夹.gitignore

1.1 把/public/添加到.gitignore

1.2 启动终端cd public

1.3 在public里面执行git init

1.4 git add .

1.5 git commit -m hello

2 到github新建一个仓库

2.1 注意：xxxxxx代表你的github用户名，这里用户名全部使用小写。

2.2 Repository name 就按照这个样式填写 xxxxxx.github.io 然后创建就行了，其他都别点。

2.3 创建后点击 SSH

2.4 复制代码git remote add origin git@github.com:xxxxxx/xxxxxx.github.io.git 到终端运行

2.5 复制代码git push -u origin master 到终端运行

3 回到github刷新页面

3.1 进入Settings 找到Github Pages 记住这个网址 https://xxxxxx.github.io/

3.2 进入config.toml 基本配置

3.3 把baseURL = “ https://www.example.com/ ”这个网址改为你的Github Pages里显示的网址baseURL = “ https://xxxxxx.github.io/ ”

4 打开终端进入这个目录 cd xxxxxx.github.io-creator

4.1 重新hugo

4.2 cd public

4.3 git add .

4.4 git commit -m hi

4.5 git push -u origin master

5 再次进入 Github Pages 这个网址 https://xxxxxx.github.io/ 就可以了
